HTML 기본 태그

1. Heading tag
(h1~6)

2. Paragraph tag
(p,br,pre)

3. Formatting tag
(b,stong,i,em,mark,del,ins,sup,sub)

4. Quotation tag
(q,blockquote,abbr,address)

5. Entity tag
(&엔티티 이름 혹은 숫자)

6. style tag
(간단한 스타일)

7. color tag
(색상 표현법)

8. background tag
(배경에 이미지 불러오기)

9. Link tag
(a tag 및 blank, self, parent, top, 지정frame이름)

10. image tag
(이미지의 각종 태그)

11. List tag
(ul(li) , ol(li) , sdl(dt))

12. table tag
(table (tr, th, td 및 collapse, colspan, rowspan))

13. block & inline tag
(block type 과 inline type)

14. iframe tag
(inline frame 에 관하여)

15. layout tag
(div , HTML5 , table을 이용한 layout)

16. form & input tag
(form과 input에 관련된 요소들)

17. input element.html tag
(input에 관련된 요소들)

18. HTML & CSS.html tag
(css 임포트 방식 및 선택자 )

19. HTML & javascript tag
( object 기반의 웹의 동작 구현 )

HTML5 기본 태그

1. sementic tag
(header, nav, section, footer 등등 추가 의미 요소)

2. input tag
(html5 에 추가된 input 요소)

3. input type tag
(input name에 관한 여러 요소들)

4. input property tag
(input 요소의 여러 속성값들)

5. video tag
(동영상 플레이 관련)

6. audio tag
(오디오 플레이 관련)

7. plugin tag
(flash player, pdf등 object 관련)
