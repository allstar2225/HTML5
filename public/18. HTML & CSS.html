- inline 방식 -

<p style="속성명 : 속성값"> 내용 </p>

- internal 방식 -

<head>
<style>
body { 속성명 : 속성값" }
</style>
 </head>

- external 방식 -

<head>
<link rel="stylesheet" href="css파일경로">
</head>

- HTML 요소 선택자 -

p { color: red; font-size: 14px; }
<p> 내용 </p>

- ID 선택자 -

<style>
#ID선택자이름 { color:red }
</style>
<p id="ID선택자이름"> 자바스크립트와 혼동될수 있으므로 되도록 클래스를 쓰는게 좋긴하다 </p>

- 클래스 선택자 -

<style>
.클래스선택자이름 { color:red }
</style>
<p class="클래스선택자이름"> 특정 집단의 여러요소를 한번데 선택 가능. </p>
<p class="클래스선택자이름"> 내용 </p>
<p class="클래스선택자이름"> 내용 </p>
