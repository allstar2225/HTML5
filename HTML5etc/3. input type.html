- button -

+ 단순 버튼

- number -

+ 텍스트 입력칸에 숫자를 조정하는 버튼이 생성됨

- range -

+ 막대기로 숫자 조절이 가능하게 함

( 0 100 )

- color -

+ 색상을 클릭하여 지정하도록 함

- date -

+ 날짜를 클릭하여 지정하도록 함

- time -

+ 시간을 입력

- datetime-local -

+ 날짜와 시간을 입력

- month -

+ 연도와 월 입력

- week -

+ 연도와 주 입력 입력

- email -

+ email 입력칸 생성. 이메일 양식에 맞추지 않으면 전송 안됨.

- URL -

+ url 주소를 입력하게 해주는 칸

- TEL -

+ 전화번호 입력칸

- search -

+ 검색 입력칸
